# almoxarifado.3b
sistema de almoxarifado para o SENAI feito pela turma Tercetroia

## Objetivos do projeto:
Este sistema foi desenvolvido na disciplina de Implantação de Sistemas do SENAI. O objetivo é controlar o estoque de materiais de TI e escritório da instituição., registrando, saídas e gerando alertas de estoque baixo.
